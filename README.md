## Overview

NetTrack transforms your Netflix experience into a social activity. Share what you're watching, discover what your friends are enjoying, explore top-rated movies and TV shows, and chat in real-time in a single platform. 

## Features

### Netflix Activity Sharing
* Track and share your Netflix viewing history with friends
* See what your friends are currently watching
* Discover trending content within your social circle

### Friend Management
* Add and manage friends
* View friends' profiles and activity
* Remove connections as needed

### Top Rated Content
* Browse top-rated Netflix movies and TV shows
* Get detailed information including reviews and ratings
* Filter content by favorite genres

### Real-Time Chat
* Instant messaging with friends
* Live chat updates using WebSockets
* Discuss shows and movies in real-time

### User Profiles
* Customizable profile with avatar upload
* Set favorite genres
* Track your viewing history
* Manage account settings

## Tech Stack

* **Languages:** JavaScript, CSS, HTML
* **Frontend:** React
* **Styling:** Tailwind CSS
* **Backend:** Firebase
* **API:** TMDb API (The Movie Database - for Netflix content data and reviews)
